# 自动驾驶三维目标检测模型
（LiDAR + 神经网络）
基于激光雷达点云数据，结合深度学习神经网络实现自动驾驶场景下的三维目标检测，可对车辆、行人、骑行者等典型交通目标进行类别识别与 3D 框定位。
## 项目简介
本项目面向自动驾驶环境感知任务，以激光雷达（LiDAR）点云为主要输入数据，构建并训练深度神经网络模型，完成三维目标检测任务。旨在为自动驾驶车辆提供可靠的周围目标位置、尺寸、朝向及类别信息，支撑路径规划、决策控制等上层模块。
## 主要任务
从激光雷达点云中提取有效特征
基于神经网络实现 3D 目标检测与分类
输出目标类别、3D 包围框、置信度等结果
支持自动驾驶典型场景推理与评估
## 环境依赖
Python 3.x
PyTorch / TensorFlow（二选一即可）
Open3D / Mayavi / PyVista 点云处理库
NumPy、Pandas、Matplotlib
CUDA（可选，用于 GPU 加速训练）
## 数据集
项目可适配主流自动驾驶 3D 检测数据集：
KITTI 数据集
nuScenes 数据集
自定义激光雷达采集点云数据
## 数据预处理
点云去噪、滤波
坐标转换与坐标系统一
生成 3D 标注与训练样本
## 模型结构
本模型以点云深度学习网络为核心，典型结构包括：
点云特征提取模块
PointNet / PointNet++ / VoxelNet 等特征编码
3D 候选框生成模块
基于体素化或原始点云生成 proposals
检测头与回归模块
目标分类
3D 框回归（中心、尺寸、旋转角）
后处理
NMS 非极大值抑制
置信度筛选
## 功能说明
### 数据加载与可视化
读取激光雷达 .bin / .pcd 格式点云
可视化点云与 3D 检测框
### 模型训练
支持单卡 / 多卡训练
配置学习率、epoch、batch size 等超参
### 模型评估
计算 mAP、Precision、Recall
针对车辆、行人、Cyclist 分别评估
### 实时推理
输入激光雷达帧，输出 3D 检测结果
可接入自动驾驶仿真或实车平台
