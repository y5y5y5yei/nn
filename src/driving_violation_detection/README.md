# 交通违章检测系统

基于 CARLA 自动驾驶仿真平台，构建交通标志检测数据自动采集与违章检测系统。

## 功能概述

- 基于 CARLA 仿真平台的自动驾驶交通场景模拟
- 交通标志检测数据自动采集
- 违章行为检测与记录

## 项目说明

该程序基于 CARLA 自动驾驶仿真平台，构建了一个交通标志检测数据自动采集系统。后续功能待扩充，目前完成的是自动驾驶的功能。

## 环境要求

- CARLA 0.9.11
- Python 3.7

## 快速开始

1. 下载并安装 CARLA（推荐 0.9.11）
2. 确保 Python 版本为 3.7（与 CARLA 0.9.11 匹配）
3. 将 CARLA 模拟器部署在与项目 `nn` 同级的文件夹下，确保 egg 文件路径正确
4. 运行主程序：

```bash
python main.py
```
