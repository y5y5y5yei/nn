# 多模态机器人导航系统 (Multimodal Robot Navigation System)

基于多传感器融合的机器人自主导航系统，利用深度学习和强化学习实现复杂环境中的智能导航。

## 项目简介

本项目实现了一套多模态感知与导航系统，融合视觉、激光雷达和IMU等多源传感器数据，通过注意力机制进行特征融合，结合深度强化学习实现机器人在复杂环境中的自主导航。

## 环境要求

- Python 3.8+
- PyTorch
- CARLA 模拟器（可选）

## 快速开始

```bash
pip install -r requirements.txt
python main.py
```
