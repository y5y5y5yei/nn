# 无人机感知模块

基于深度学习的无人机图像感知与分类模块，实现无人机视觉感知功能。

## 模块说明

`image_classification` 为图像分类模块，`laterpicture` 为在有一个训练图片下模型分类的运行截图。

## 环境要求

- Python 3.6+
- PyTorch 和 torchvision
- NumPy
- PIL (Pillow)
- scikit-learn
- matplotlib
